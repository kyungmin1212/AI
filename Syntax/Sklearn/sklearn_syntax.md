    - `train_test_split`
    - `cross_val_score`
    - `GridSearchCV`
- `from sklearn.linear_model import ~`
    - `LinearRegression`
    - `LogisticRegression`
- `from sklearn.metrics import ~`
    - `mean_squared_error`
    - `accuracy_score`
- `df` : DataFrame 이름 / `sr` : Series 이름
